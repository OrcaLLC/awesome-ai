# Contributing

This repository accepts fictional entries only.

## Scope

- Add made-up projects, libraries, tools, or services related to AI.
- Keep naming and descriptions realistic and professional.
- Do not add satire, parody, or overt jokes.

## Hard Rules

- Never add real repositories, organizations, products, papers, people, or companies.
- Never copy existing project names from public ecosystems.
- Never use real-world domains.
- Use clearly fictional domains and vary them across entries (for example: `.girthhub`, `.squanchbase`, `.nebulanet`).
- Every description must be original and concise.

## Entry Format

Use this structure:

- `[ProjectName](https://project-name.girthhub) - One-line description.`

## Pull Request Checklist

- I confirm all new entries are fictional.
- I confirm all links use fictional domains only.
- I confirm I did not copy names from real projects.
- I confirm descriptions are original.
